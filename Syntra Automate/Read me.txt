Syntra - README

Requisitos:
- Ubuntu 20.04/22.04/24.04
- sudo/root

Passos rápidos (local dev):
1. Backend
$ cd backend
$ python3 -m venv venv
$ venv/bin/pip install -r requirements.txt
# create .env with DATABASE_URL and keys
$ venv/bin/uvicorn app.main:app --reload --port 8000

2. Frontend
$ cd frontend
$ npm install
$ npm run dev

API principais:
- POST /auth/token -> OAuth2 password form (username,password) -> retorna access_token
- POST /mikrotiks -> criar (admin)
- GET /mikrotiks -> listar
- POST /exec/run -> executar (campos: targets [ids], command)

Variáveis de ambiente:
see /backend/.env.example

Instalação automatizada:
sudo ./scripts/install_syntra.sh

Arquitetura:
- FastAPI backend (uvicorn), Postgres, frontend React Vite served via nginx
- SSH via asyncssh com concurrency semaphore
